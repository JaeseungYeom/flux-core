flux_kvs_get_namespace(3)
=========================
:doctype: manpage


NAME
----
flux_kvs_get_namespace - get KVS namespace


SYNOPSIS
--------
 #include <flux/core.h>

 const char *flux_kvs_get_namespace (flux_t *h);

DESCRIPTION
-----------

`flux_kvs_get_namespace()` will determine the current namespace to
use.  It checks if the environment variable FLUX_KVS_NAMESPACE is set.
If it isn't, the default namespace of "primary" will be returned.


RETURN VALUE
------------

`flux_kvs_get_namespace()` returns namespace on success, or NULL on
failure.


ERRORS
------

EINVAL::
One of the arguments was invalid.
